## seifEddine.github.io

Personal portfolio and blog site built with Jekyll using the Minimal Mistakes theme.

### Live Site
https://seifEddine.github.io

### Local Development
1. Install Ruby and Bundler.
2. Install dependencies:

```sh
bundle install
```

3. Run the site:

```sh
bundle exec jekyll serve
```

Open http://127.0.0.1:4000 in your browser.

### Tech Stack
- Jekyll
- Minimal Mistakes theme

### Notes
- Site configuration is in `_config.yml`.
- Navigation is in `_data/navigation.yml`.
- Content lives in `_pages`.
